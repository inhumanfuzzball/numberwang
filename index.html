A maths puzzle game.

How to play

  1. Use the numbers 1 to 9 to complete the equations.
  2. Each number may only be used once.
  3. Each row is a maths equation. Each column is a maths equation.
  4. All calculations are perfromed left to right (do not obay order of operation).
